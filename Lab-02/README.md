## 📖 Overview

This laboratory explores one of the fundamental concepts of Artificial Intelligence: **Simple Reflex Agents**.

A Simple Reflex Agent makes decisions solely based on the current percept, without considering historical information or future consequences. These agents follow predefined condition-action rules to respond intelligently to environmental changes.

This lab demonstrates how AI agents can automate real-world tasks through rule-based decision making.

---

## 🎯 Learning Objectives

By completing this lab, students will:

- Understand the architecture of Simple Reflex Agents
- Learn how condition-action rules work
- Design intelligent systems for real-world scenarios
- Implement AI agents using Python
- Analyze agent behavior in dynamic environments

# 🧠 Implemented Tasks

## 🚨 Task 1 – Fire Alarm System Agent

### Scenario

The agent continuously monitors:

- Smoke Levels
- Temperature

### Agent Actions

| Condition | Action |
|------------|---------|
| Smoke Detected | Sound Alarm + Activate Sprinklers |
| High Temperature | Sound Alarm + Call Fire Department |
| Normal Environment | No Action Required |

### AI Concept

This system demonstrates how a Simple Reflex Agent reacts instantly based on environmental conditions.

---

## 🌱 Task 2 – Automatic Watering System Agent

### Scenario

The agent monitors:

- Soil Moisture Level

### Agent Actions

| Condition | Action |
|------------|---------|
| Dry Soil | Turn ON Water Pump |
| Moist Soil | Turn OFF Water Pump |

### AI Concept

This agent automates plant care using predefined decision rules.

 Actions
